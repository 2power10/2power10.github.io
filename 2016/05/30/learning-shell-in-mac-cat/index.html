基本

cat基本用途是把指定文件读取出来，并输出到STDOUT中。

cat [-benstuv] [file ...]

常用功能

-b 打印出行数，不包含空白行，从1开始
-n 打印出行数，包括空白航，从1开始
-e 显示不能打印的字符，每行结尾打印$符号
-t 显示不能打印的字符，制表符显示为^I

使用场景

打印一个文件到终端来查看其中内容

cat file

打印文件到另一个文件中

cat file1 > file2

打印文件到另一个文件结尾

cat file1 >> file2
